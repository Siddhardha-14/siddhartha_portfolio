TYPOGRAPHY:
- Display / Headings: Poppins Bold
  Used for screen titles, parking lot names
- Body: Inter Regular / Medium
  Used for descriptions, labels, form fields
- Micro-copy: Inter Light
  Used for captions, hints, timestamps
Reasoning: Poppins feels modern and trustworthy 
for a mobility app. Inter ensures readability 
at small sizes on mobile.

COLOUR PALETTE:
Primary Background: #1A1A2E (Deep Navy)
  — Creates a premium, night-mode feel suitable 
    for in-car/low-light usage
Secondary Surface: #16213E
  — Cards, modals, bottom sheets
Accent / CTA: #6C63FF (Violet/Indigo)
  — Buttons, selected states, highlights
Success Green: #4CAF50
  — Available slot indicators
Error Red: #FF5252
  — Occupied slot indicators
Selected State: #7B68EE
  — Active booking selection
Text Primary: #FFFFFF
Text Secondary: #A0AEC0

Reasoning: Dark navy reduces eye strain for 
travellers using the app at night or in bright 
sunlight (high contrast). Violet accent is 
distinctive in the mobility app space.

COMPONENTS:
- Primary Button (filled violet, rounded)
- Secondary Button (outlined)
- Parking Slot Card (3 states: available/occupied/selected)
- Info Card (rate, distance, amenities)
- Bottom Sheet / Modal
- Map Pin component
- QR Code display card

ICONOGRAPHY:
- Rounded, filled icon style
- Categories: Navigation, Parking, Payment, 
  Vehicle type, Amenities

SPACING SYSTEM:
Base unit: 8px
Padding scale: 8 / 16 / 24 / 32px
